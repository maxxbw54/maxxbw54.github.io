A last-minute checklist for *Conference* paper submission

  1. Ensure that the information (e.g., title, authors, abstract) in the submission systems (like HotCRP & EasyChair) matches the infomation in the paper.
  2. Is the title good enough?
  3. Search common reference errors/typos in pdf file, like "??", "[?]", "..", double space, etc.
  4. Convert pdf to word format and run Grammarly.
  5. Check latex template if follow the submission requirement, for example, ACM conferences may require to use "\documentclass[sigconf,review,anonymous]{acmart}".
  6. Check if already added the link to the anonymized replication package.
  7. Double-check the topic of the paper in the system with your co-authors/mentor/supervisor.
  8. Double-check the co-authors' conflict-of-interests (COIs).
  9. Check the reference if it contains the right/latest version of all the citations, e.g., replace the Arxiv version with the accepted version if applicable.
  10. Check tense by following Using tenses in scientific writing.

A last-minute checklist for *Journal* paper submission

  1. Double check the table/fig/equation/etc numbers mentioned in the response are synced with the original paper.
  2. Index all the comments in the letter to be clear and easy to follow for the editor and reviewers.
  3. For the response to the meta-review, provide a mention of the reviewer comments that are related to it, e.g., (Please also see the response to Comment R1.X, R2.Y, etc.) So that the reviewers can easily cross reference if he/she wants to.
  4. Standardise the formatting in the response letter to be uniform.
  5. Ensure that the cover letter appears BEFORE the paper in the Proof file when you submit it.
  6. Recommend AE.
  7. Back up the Proof file.

Good luck! 🤞
